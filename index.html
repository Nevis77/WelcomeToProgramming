Programming refers to a technological process for telling a computer which tasks to perform in order to solve problems. You can think of programming as a collaboration between humans and computers, In which humans create instructions for a computer to follow (code) in a language computers can understand

There are primarily four types of programming, imperative, functional, logical and object-oriented. Alternative names and other primary types may include procedural, scripting and database programming

Programming in a sense, is a way by which humans communicate to computers from a higher level language such as any spoken, most popularly English, to a lower level language that the computer understands, that is machine language or more popularly known as binary.

With the age of the internet, any one person can begin their journey to become a programmer. However, its not as simple as hoping on YouTube and binging tutorials in hopes to understand everything as if learning by diffusion. Practice is key as you teach yourself how to implement what you've learnt while building muscle memory, enabling you to easily recall the context within which that particular principle will be applied.
Programmers are, in a nutshell, problem-solvers whose main job is to essentially breathe their thoughts into life through code. If you wanted to implement a simple calculator that would perform basic calculations, it is easy to think how it would work (especially if you've been through formal education) but actually implementing it may prove a challenge especially to newer programmers just starting out.
This therefore shows us that programmers also need other skills in order to be truly a problem solver. Some of these skills include:

  1. Critical thinking.
  2. Open-mindedness.
  3. Problem-Solving.
  4. Teamwork.
  5. Communication Skills.
  6. Patience.
  7. Adaptability.
  8. Self-Motivation.
  9. Discipline.
